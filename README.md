# Rare plants page

## install dependecies
```
npm i
```
## fill env file in server side
```
SERVER_DOMAIN=
SERVER_PROTOCOL=
SERVER_PORT=

IMG_FOLDER=
ROOT_DIR=

DB_CONNECTION= 
TOKEN_SECRET=
```
## fill env file in client side
```
REACT_APP_SERVER_ADDRESS=
```

## launch server
```
npm run server
```

## launch client 
```
npm run start
```
